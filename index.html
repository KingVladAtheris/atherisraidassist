Raid organization tools for World of Warcraft Classic guilds — Discord bot for seamless signups and reserves • In-game addon for assignments and tactical maps.

Watch YouTube Tutorial

Discord Bot

  • • Admins create, close raids and finalize rosters, pull export strings and can add import strings.
  • • Soft reserves functionality: set limits per role and pull up items instantly with autocomplete.
  • • Signups are quick – just open the modal, choose your class/spec and status, and it's locked in right away.
  • • The embeds stay updated live, showing attendance, reserves, and the locked roster clearly.
  • • It ties in smoothly with the addon through easy import/export strings.

In-game Addon

  • • Import, export or create raids. Add or import raiders. Add encounters and set assignments.
  • • Item tooltips show soft reserves live in game so you don't have to check outside lists and can distribute loot quickly.
  • • The map editor supports all classic raids, Vanilla through WotlK. Add raider markers or symbol markers for easy fight coordination.
  • • Encounter presets let you set up raids quickly and then broadcast them to all your group.
  • • Export raid strings and share them using the Discord bot for proper planning and coordination.
