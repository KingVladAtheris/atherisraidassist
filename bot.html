Discord Bot

Raid Creation

Channel Admins / Raid Leaders: start with /raid create

• Set raid name
• Set exact slot numbers per role (tanks / healers / dps)
• Optional: soft-reserve limits per role (default = 0)

→ Bot posts a live updating embed showing all signups in real time

Player Signups & Soft Reserves

Players click Manage Signup button:

→ Enter character name (must match in-game exactly)
→ Choose class + spec + status (attend / maybe / absent)

If soft reserves are enabled:
Use commands /softreserve add • list • remove • clear

Close Raid & Finalize Roster

When ready → click Close Raid

Then:
• Click players to finalize each role (green = in roster, blue = not yet)
• Click Confirm Roster when finished
• You can re-open and change the roster later if needed

Publish Raid

After confirming roster → Publish Raid button appears.

Useful actions:
• Export Raid / Export Roster string → import into addon
• Publish Raid → shows nice final summary embed for everyone
• Upload Import String (.txt file) — final raid data from addon back to bot
• After upload → Download Import String becomes available for all members
